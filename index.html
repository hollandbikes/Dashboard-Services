1. Location Longue Durée (LLD)

1.1 GreenLeaze

GreenLeaze est notre prestataire pour la LLD. GreenLeaze nous achète le vélo et le loue au client pour un prix mensuel prédéfini. Le contrat est entre le client et GreenLeaze. Ce système remplace MyPangee.

Avantages pour le client :

  • Accessibilité (paiements mensuels).
  • Sérénité (service tout compris).
  • Possibilité de changer de vélo régulièrement.

Forfait révision mensuel (Option) :

Ces montants ne sont pas intégrés dans le loyer de base de GreenLeaze. L’atelier doit facturer cette prestation séparément à GreenLeaze.

  • Musculaire : 6,50€/mois
  • Électrique : 8,00€/mois
  • Cargo : 10,50€/mois

Procédure de vente Magasin

1. Création de la simulation : Saisir le montant TTC (Vélo + Accessoires + Éco-part). Choisir la durée (12, 24 ou 36 mois) et ajouter l'entretien si besoin.

2. Saisie des informations client : Remplir les coordonnées de l’utilisateur. Le client reçoit un lien par email/SMS pour valider son dossier via l'open banking.

3. Livraison du matériel : Une fois le statut "Signé", cliquer sur "Livrer le matériel" sur le dashboard GreenLeaze pour déclencher le paiement.

Facturation CycleSoftware

Facturer à : Greenleaze SAS, 21 boulevard de Beauséjour, 75016, Paris (RCS Paris 984942573).

Mentions obligatoires sur la facture :

  • Modèle du vélo et prix TTC.
  • N° Vélopass.
  • Accessoires inclus.

Ligne Commission Vendeur (2%) :

Calcul = Durée (mois) × 2 % × (loyer TTC / 1,2)

3. Assurance

3.1 Assurance 4 mois offerte

Holland Bikes offre les 4 premiers mois d'assurance VOL. Elle couvre le vol par effraction, agression ou à l'extérieur si attaché à un point fixe avec un antivol agréé. Franchise 10% (min 50€).
